Detalhes da versão

Informações da release atual do Gerador de Documentos.

Versão 1.0

Primeira versão operacional do sistema de geração de documentos com preenchimento automático de variáveis e exportação de arquivos.

Resumo da versão

Versão 1.0
Data da versão 21/04/2026
Status Produção inicial
Responsável Equipe Transposeg

Funcionalidades entregues

  • Cadastro e edição de variáveis por documento
  • Leitura e pré-visualização de arquivos DOCX
  • Preenchimento automático de placeholders no modelo
  • Geração de DOCX preenchido para download
  • Organização por abas para diferentes grupos de variáveis

Melhorias previstas

  • Validação automática de variáveis obrigatórias
  • Template de documentos múltiplos
  • Área administrativa para manutenção de modelos

Sugestão de versionamento

  • Para os documentos modelo versionar (v 1.0)
  • Caso houver novos parâmetros versionar (V 1.1)
  • Caso houver modificações textuais no documento versionar (V 2.0)

Base de dados

  • Quando abrir a aplicação pela primeira vez cria o diretório em (C:\Transposeg\Gerador\data.json)
  • Não modificar os nomes dos diretórios
  • Não modificar o nome do arquivo data.json
  • Se houver novos parâmetros substituir o data.json atual pelo novo que estará em uma pasta compartilhada

Observações

Esta página serve como referência funcional da release atual. Em versões futuras, pode ocorrer de mudar ou acrescentar novas instruções e etc. .
